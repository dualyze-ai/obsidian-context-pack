# LinkBox — Overview

A simple browser-based bookmark manager. Paste a URL, add a tag, save. No backend, no login.

## What it does
- Save links with a title and optional tag
- Search saved links by keyword or tag
- Delete links you no longer need

## What it doesn't do
- No folders, no nested tags
- No import from browsers
- No sharing
