# 近代絵画

19世紀末から20世紀にかけて花開いた多様な絵画運動の総称。印象派の「見たままを描く」という革新をさらに推し進め、形・色・概念そのものを問い直した。

## 主な運動

| 運動 | 特徴 |
|---|---|
| ポスト印象派 | 個人の感情・構造・象徴を重視 |
| キュビスム | 複数の視点を同一画面に描く |
| シュルレアリスム | 夢・無意識の世界を視覚化 |
| 表現主義 | 感情の激しい表現を優先 |

## 代表的な画家

- [[フィンセント・ファン・ゴッホ]] — 渦巻く筆致と鮮烈な色彩。「星月夜」「ひまわり」。
- [[パブロ・ピカソ]] — キュビスムを創始した20世紀最大の革命児。
- [[サルバドール・ダリ]] — シュルレアリスムの代表格。「記憶の固執」。
